PROGETTI

In questa sezione sono presenti tutti i progetti realizzati lungo il mio percorso formativo.

Digital Donation

Digital Donation è un sistema centralizzato sulla creazione di una piattaforma digitale che permette di semplificare e velocizzare la gestione delle donazioni rendendo più rapide le procedure di registrazione dei donatori e prevedendo un sistema più efficiente per l’organizzazione delle sedute.
Inoltre, il nuovo sistema è volto anche agli operatori delle sedi locali, in modo da alleggerire l’effort del personale interno che potrà così dedicarsi con maggiore impegno a mansioni più consone al proprio ruolo. La piattaforma è stata realizzata con il framework Springboot Java, HTML, CSS, Javascript e JSTL. Nel link di riferimento è presente tutta la documentazione.

PCMasterRace

PCMasterRace è una piattaforma e-commerce dedicata alla vendita di componenti per PC e non solo.
Il sito è stato progettato ed implementato secondo architettura Model-View-Control.
Le classi Model che rappresentano i bean, tramite un approccio Data-Source vanno ad interfacciarsi con il Database MySQL.
Le classi Control realizzate con Java Servlet rappresentano la parte back-end del sito, qui vengono effettuati i controlli sulle operazioni effettuate dall'utente e tramite le classi Model si interagisce con la base dati.
Infine la View rappresenta tutte le pagine che vengono mostrate a video all'utente e sono state realizzate con HTML, JSP e CSS, tutte le pagine del sito sono responsive.
I controlli sui form sono stati realizzati con Javascript e JQuery, i controlli per l'operazione di registrazione di un nuovo utente sono stati realizzati tramite AJAX con codice JSON.

Food Delivery DataBase

Il progetto consiste nella realizzazione di un Database che possa gestire gli ordini e le consegne a domicilio dei ristoranti durante l'emergenza scaturita dalla pandemia Covid-19.
La prima fase del progetto si centralizza sulla progettazione concettuale, che si divide in descrizione del problema iniziale, analisi dei requisiti e specifica dello schema Entità-Relazioni. La seconda parte del progetto riguarda la progettazione logica, sulla base del carico applicativo vengono stipulate la tavole degli accessi e delle operazioni che permettono un analisi delle prestazioni e delle ridondanze.
Tramite l'eliminazione delle gerarchie e l'analisi di attributi composti e multivalore otteniamo uno schema ristrutturato che può essere mappato in uno schema relazionale.
Infine è possibile creare il Database con il Data Definition Language e popolarlo di dati tramite il Data Manipulator Language.
La parte finale consiste nell'integrazione di alcune query in Java tramite il connettore JDBC.
